# FFC Reality Checker

A collection of reality checks, countdowns and documented predictions.

## Project structure

```text
index.html              Overview
css/                    Stylesheets
js/                     JavaScript
img/                    Images
reality-checks/         Individual reality check pages
```

## Naming convention

Images are named using the following pattern:

```
<page-name>--<description>.<ext>
```

Examples:

```
fuellmich--portrait.png
umbele-town--telegram-post.png
```
